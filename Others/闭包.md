# 闭包

2. 闭包的原理可不可以说一下？
3. 你是怎样使用闭包的？

```javascript
function fun(n, o) {
  console.log(o);
  return {
    fun: function(m) {
      return fun(m, n);
    }
  };
}
var a = fun(0);
a.fun(1);
a.fun(2);
a.fun(3);
var b = fun(0)
  .fun(1)
  .fun(2)
  .fun(3);
var c = fun(0).fun(1);
c.fun(2);
c.fun(3);
/*
undefined
0
0
0

undefined
0
1
2

undefined
0
1
1

*/
```
